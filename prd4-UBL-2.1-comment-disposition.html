1. Overview

The fourth public review of OASIS UBL 2.1 http://docs.oasis-open.org/ubl/prd4-UBL-2.1/UBL-2.1.html was announced May 23, 2013: https://lists.oasis-open.org/archives/ubl/201305/msg00022.html.

This document details the disposition of comments received both publicly and from within the committee.

2. Non-material public comments

The following non-material public comment was received.

2.1. Namespace and schema questions

  • Received June 7, 2013 from Ole Madsen

  • http://lists.oasis-open.org/archives/ubl-comment/201306/msg00000.html

  • Withdrawn June 10, 2013 by Ole Madsen

  • http://lists.oasis-open.org/archives/ubl-comment/201306/msg00001.html

  • Summary: The software in use is non-conformant with respect to the presence of an internal declaration subset in the XML document. There are no problems with the UBL schemas when using conformant software.

Disposition: No action.

3. Non-material committee comments

The following non-material committee comment was received.

3.1. Published non-normative citation

  • http://lists.oasis-open.org/archives/ubl/201306/msg00005.html

Disposition: modify the citation and add the citation to the list of non-normative references

4. Non-material TC Administration comments

The titles and URI addresses for the latest version of the cited committee notes are incorrect in PRD4:

Universal Business Language Version 2.1 Abstract Syntax Notation 1 (ASN.1) Alternative Representation Committee Note 01.
http://docs.oasis-open.org/ubl/UBL-2.1-ASN.1/v1.0/UBL-2.1-ASN.1.html
Universal Business Language Version 2.1 Unified Modeling Language (UML) Alternative Representation Committee Note 01
http://docs.oasis-open.org/ubl/UBL-2.1-UML/v1.0/UBL-2.1-UML.html

Disposition: modify the titles and citations as follows per instructions from TC Administration:

UBL 2.1 Abstract Syntax Notation 1 (ASN.1) Alternative Representation Version 1.0
http://docs.oasis-open.org/ubl/UBL-2.1-ASN.1/v1.0/UBL-2.1-ASN.1-v1.0.html
UBL 2.1 Unified Modeling Language (UML) Alternative Representation Version 1.0
http://docs.oasis-open.org/ubl/UBL-2.1-UML/v1.0/UBL-2.1-UML-v1.0.html
